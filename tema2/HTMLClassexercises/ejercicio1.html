HTML

Introducción

HTML, siglas en inglés de HyperText Markup Language (‘lenguaje demarcas de hipertexto’), hace referencia al lenguaje de marcadopara la elaboración de páginas web. Es un estándar que sirve dereferencia del software que conecta con la elaboración de páginasweb en sus diferentes versiones, define una estructura básica y uncódigo (denominado código HTML) para la definición de contenido deuna página web, como texto, imágenes, videos, juegos, entre otros.

Primeras especificaciones

Tim Berners-Lee en 1991 describe 18 elementos que incluyen eldiseño inicial y relativamente simple de HTML. Trece de estos elementos todavía existen en HTML 4.
